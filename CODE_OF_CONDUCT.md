# Code of Conduct

## Our Pledge

This is a scientific research project. We pledge to make participation 
a harassment-free experience for everyone, regardless of background.

## Our Standards

**Positive behavior:**
- Respectful scientific discourse
- Constructive criticism
- Focus on theory and evidence
- Collaboration over competition

**Unacceptable behavior:**
- Personal attacks
- Trolling or insulting comments
- Harassment of any kind
- Publishing others' private information

## Enforcement

Violations can be reported by opening an issue or contacting the maintainer.

## Scope

This Code of Conduct applies within all project spaces (GitHub, Discord, etc.)
